What you need
Claude Pro subscription — $20/month. Go to claude.ai/pro to subscribe.
A computer — Mac, Windows, or Linux. Claude Code runs on your computer, not in a browser.
Node.js (LTS) — required to install Claude Code via npm. Download from nodejs.org if you don't have it. Run node -v in Terminal to check.
Git — for cloning the template. Pre-installed on Mac and Linux. Windows: download from git-scm.com. Run git -v in Terminal to check.
1
Install Claude Code
This is the app that runs your carousel factory. It's a command-line tool — think of it as a very smart terminal assistant.
1. Open your computer's Terminal app (Mac: Terminal, Windows: PowerShell)
2. Paste this command and press Enter:
npm install -g @anthropic-ai/claude-code

First time using Terminal? On Mac: Cmd+Space → "Terminal" → Enter. On Windows: Win → "PowerShell" → Enter. Then paste the command above.

"command not found"? You need Node.js first. Download Node.js here — click LTS, install, restart Terminal, try again.

After installation, type claude and follow the login prompts. It will open Claude Code.
2
Get the Template
Clone the repo, enter the folder, then open Claude Code inside it.
git clone https://github.com/Sophiekwon-syd/carousel-automation-template.git
cd carousel-automation-template
claude
Run these three commands in Terminal, one at a time, pressing Enter after each. The last command opens Claude Code inside your project folder — this is important, as Claude reads CLAUDE.md from the current directory.

First time logging in? Claude Code will prompt you to authenticate with your Claude Pro account. Follow the link it shows you.

3
Set Up Your Brand
Copy the example files and replace placeholders with your information. You only do this once.
1. Run these three commands in Terminal to copy the example files:
cp config.example.json config.json
cp tone-guide.example.md tone-guide.md
cp topic-memory.example.json topic-memory.json
2. Then open Claude Code (claude) and tell it your brand details:
Edit config.json with my brand info: brand.name is "[Your Brand Name]", brand.account is "@yourhandle", content.niche is "[what your account covers]", content.target_audience is "[who reads your content]".

Also update tone-guide.md to match this voice: [describe your tone in a sentence or two].

Or just describe yourself naturally: "I run a Korean parenting account in Australia called @aussie.umma. Make the config match." — Claude will edit the files for you.

4
Generate Carousels
One command. That's it. Claude does everything automatically — researches topics, writes copy, builds HTML, validates, and creates PNG screenshots.
claude /daily-run
First run takes about 15–20 minutes. Approve each permission prompt when Claude asks — web search, file writes, and the screenshot tool are all normal.

"Allow" or "Yes" to everything? Claude will ask to: (1) search the web, (2) create files, (3) run the screenshot tool. All normal — approve each one.

5
Find Your Output
Everything is saved in a folder named with today's date.
outputs/
└── 2026-05-08/ ← today's date
    ├── topic-slug-one.html
    ├── topic-slug-two.html
    ├── research_1.json
    ├── research_2.json
    ├── outlines.json
    ├── copy_1.json
    ├── copy_2.json
    ├── run-log.json
    └── images/
         ├── topic-slug-one-01.png
         ├── topic-slug-one-02.png
         ├── ...
         └── topic-slug-two-10.png
Upload the PNGs to Instagram in order — each carousel has 10 cards numbered 01 through 10.
6
Verify the Output
After the run completes, confirm the pipeline worked end-to-end before setting up automation.
1. Check that outputs/YYYY-MM-DD/images/ contains PNG files — 10 per carousel.
2. Open one PNG and confirm the card dimensions are 1080×1350px and the design matches examples/carousel-example.html.
3. Check outputs/YYYY-MM-DD/run-log.json — "status": "complete" means a clean run.

PNGs missing? Run npx puppeteer browsers install chrome in Terminal, then run /daily-run again.

7
Automate Everything
The automation file is already included in this template. You just need to give GitHub your API key so it can run while you sleep.
1. Go to your repository on GitHub.com.
2. Click Settings (top tab) → Secrets and variables → Actions.
3. Click New repository secret.
4. Name: ANTHROPIC_API_KEY
5. Secret: (Paste your Claude API key)

Success: Once the key is added, your carousels will generate automatically at 2:30am KST every day.

Setup complete. Your carousels will now generate automatically every day.

Common issues
"npm: command not found"
Download Node.js (LTS version), install it, close Terminal, reopen it, then try Step 1 again.
"permission denied"
Mac: System Settings → Privacy & Security → Full Disk Access → add Terminal. Windows: right-click PowerShell → Run as Administrator. Linux: use sudo before the command.
Claude Code opens but nothing happens
Type /daily-run inside Claude Code and press Enter. The / tells it to run the command.
"No such file" when cloning
The GitHub URL in Step 2 might be wrong. Check that the repo name and username are correct. If you don't have GitHub, use the direct clone link from the original repo.
Carousels look wrong (no design, wrong colors)
Type /clear inside Claude Code to reset its memory, then run /daily-run again.
PNG files are empty or missing
Run npx puppeteer browsers install chrome in your Terminal first, then run /daily-run again.
Topics keep repeating
Check that topic-memory.json exists in your project folder and has at least one entry. Claude uses this file to avoid repeats.

Open examples/carousel-example.html in your browser to see what the output looks like before running. Not comfortable with Terminal? See setup-guide-easy.html for a no-command version.
