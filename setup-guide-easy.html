1
Install Claude Code
Claude Code is the app that runs your carousel factory. You install it once with a single command.
1. Open Terminal on Mac (Cmd+Space → type "Terminal" → Enter) or PowerShell on Windows (Start → type "PowerShell" → Enter).
2. Click the command below to copy it, then paste it in Terminal and press Enter.
npm install -g @anthropic-ai/claude-code
3. Wait for it to finish (about 30 seconds). Then type claude and press Enter.
4. Follow the login prompts — it will ask you to sign in with your Claude Pro account.

Once you see a > prompt inside Claude Code, you're ready for Step 2.

"npm: command not found"? Node.js is not installed yet. Download Node.js (LTS), run the installer, close Terminal, reopen it, then try the command again.

2
Set Up the Project
You are now inside Claude Code. Fill in your details in the prompt below, then copy the whole thing and paste it in. Claude will download the template and set it up for you automatically.
Paste this into Claude Code — fill in the yellow fields first
Please set up my carousel automation project. Do this in order: 1. Clone https://github.com/Sophiekwon-syd/carousel-automation-template.git into the current folder 2. Go into the cloned folder 3. Copy config.example.json → config.json, tone-guide.example.md → tone-guide.md, and topic-memory.example.json → topic-memory.json 4. Edit config.json with my brand details: - brand.name = "Your Brand Name" - brand.account = "@yourhandle" - content.niche = "What your account covers, e.g. Korean parenting in Australia" - content.target_audience = "Who reads your content, e.g. Korean mums living in Australia" 5. Tell me when it's done and confirm which folder the project is in

Approve the prompts. Claude will ask for permission to access your files and run commands. Click Yes or Allow each time — this is how it does the work for you.

Not sure what to write? Just describe yourself naturally and Claude will figure it out. For example: "I run a Korean parenting account in Australia — my handle is @aussie.umma and I post tips for Korean mums raising kids here."

3
Generate Your First Carousels
Once Claude confirms the setup is complete, paste this command. That's the whole pipeline — research, copy, design, QA, screenshots — done automatically.
/daily-run
This takes about 15–20 minutes on your first run. You'll see progress in the terminal as it works through each stage.

Keep clicking "Allow". Claude will ask permission to search the web, create files, and run the screenshot tool. Every prompt is part of the pipeline — approve them all.

Don't close the window while it's running. Leave Terminal open until you see the final summary message.

4
Find Your Images
When the run completes, your PNG cards are saved here:
carousel-automation-template/
└── outputs/
    └── 2026-05-08/ ← today's date
        └── images/
            ├── topic-name-01.png
            ├── topic-name-02.png
            ├── ...
            └── topic-name-10.png
Open Finder (Mac) or File Explorer (Windows) and navigate to the images/ folder. Upload the PNGs to Instagram in order — 01 is the cover, 10 is the CTA.

Want to see what a finished carousel looks like? Open examples/carousel-example.html in your browser before running — that's the visual reference the pipeline targets.

5
Run It Every Day
From now on, generating a fresh batch of carousels takes two things: open Claude Code in the right folder, then run the command.
1. Open Terminal (Mac) or PowerShell (Windows).
2. Type this to open Claude Code in your project — copy, paste, Enter:
cd ~/carousel-automation-template && claude
3. Once Claude Code opens, type:
/daily-run
That's it. New carousels in your outputs/ folder every time.

Folder not found? Ask Claude where it saved the project in Step 2 and use that path instead. For example: cd ~/Desktop/carousel-automation-template && claude

You're set up. Run /daily-run any time you want a fresh batch of carousels.
New research. New copy. New designs. Every time.

Something went wrong?
"npm: command not found"
Node.js is not installed. Download the LTS version from nodejs.org, run the installer, close Terminal completely, reopen it, then try Step 1 again.
Claude stopped mid-way and nothing happened
It likely hit a permission prompt and waited for you. Scroll up in the terminal window — there may be a "Yes/No" or "Allow/Deny" question. Answer it and it will continue. If it's stuck with no question, type /daily-run again.
I can't find the images folder
Ask Claude directly inside Claude Code: "Where did you save the project, and where are today's PNG files?" It will tell you the exact path and you can navigate to it in Finder or File Explorer.
"permission denied"
Mac: System Settings → Privacy & Security → Full Disk Access → add Terminal.
Windows: right-click PowerShell → Run as Administrator.
PNG files are empty or missing
The screenshot tool needs a browser to be installed. Paste this in Terminal: npx puppeteer browsers install chrome Then go back to Claude Code and run /daily-run again.
The carousels are on topics I don't want
Open Claude Code in the project folder and say: "Edit config.json — add [topic] to the topics_to_avoid list." Claude will update the file so future runs skip it.
Something else went wrong
Just ask Claude. Describe what happened and it will work through it with you — that's what it's there for.
